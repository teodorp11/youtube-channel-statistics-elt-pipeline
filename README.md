# YouTube Video Analyzer

This repository contains an ongoing project developed as part of a [Udemy course](https://www.udemy.com/course/start-your-data-engineering-journey-project-based-learning). The primary goal of this application is to build a data pipeline that uses the YouTube Data API to extract video information and statistics for analysis.

> [!NOTE]
> **Work in Progress:** This project is currently under active development. The codebase, features, and this documentation will be modified and expanded as I progress through the course materials.

## Overview

- **Integration**: Utilizes the YouTube Data API for data extraction.
- **Purpose**: Educational project focused on data engineering / ELT concepts and video analysis.
